
simplay-jupyter
=====================================

Version: |release|

Animate SimPy simulations in jupyter notebook


Quickstart
----------

To get started with simplay-jupyter, install with pip::

    pip install simplay-jupyter

or with conda::

    conda install simplay-jupyter
